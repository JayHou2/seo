## Xiaohongshu post

Title:

```text
留学生出国第一个月，真正要准备多少钱？
```

Body:

```text
很多同学算留学预算时，只算 tuition 和每个月房租。

但真正容易爆预算的是“刚落地第一个月”。

因为这个月不是正常生活月，很多钱会集中一起花：

1. 押金 / 预付房租
很多租房不是只交一个月房租，可能还要押金、预付租金或申请费。

2. 临时住宿
如果宿舍或 lease 还没开始，前几晚酒店/短租也要算。

3. 单程机票和机场交通
机票、行李、机场到住处的交通，都不是小钱。

4. 床品、锅具、日用品
床单被子、锅、碗、清洁用品、转换插头，刚到时基本都要买。

5. 电话卡、交通卡、银行卡
开通本地生活的这些小项，加起来也会占预算。

6. 应急储备
刚到一个新城市，最好给自己留一笔 buffer。

我做了一个“出国第一个月启动资金计算器”，可以按城市、住宿方式、押金月数和临时住宿晚数估算。

如果你正在准备出国，建议先把第一个月的钱和正常月生活费分开算。
```
